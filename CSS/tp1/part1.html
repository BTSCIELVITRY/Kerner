Le Langage HTML

Le HTML (HyperText Mark-Up Language) est un langage dit de marquage (de structuration ou de balisage) dont le rôle est de formaliser l'écriture d'un document avec des balises de formatage. les balises permettent d'indiquer la façon dont doit être présenté le document et les liens qu'il étéblit avec d'autres documents.

Les protocoles

Le Web et le procole HTTPS Le Web ou World Wide Web est un système mettant en relation différents documents (textes, images, sons, vidéos) reliés les uns aux autres par des hyperliens. L'échange de données sur le Web est possible grâce au protocole HTTPS, HyperText Transfer Protocol. Ce dernier permet à différents appareils (un client et un serveur) ce dialogue sur la base d'un système requête-réponse. En effet, lorsque le client envoie pour la première fois une requête HTTPS à un serveur, il ouvre simultanément (sur le port SC), une connextion TCP avec ce serveur, qui peut rester ouverte pour des connections ultérieure HTTPS sert à définir la structure des messages échangés par le client et le serveur à travers la connection TCP.
